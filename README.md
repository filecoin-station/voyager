# voyager
Distributed Saturn tester as a Station module

## Development

Install [Zinnia CLI](https://github.com/filecoin-station/zinnia).

```bash
$ zinnia run main.js
```

## Release

```bash
$ ./release.sh <SEMVER>
$ # Example
$ ./release.sh 1.0.0
```

Use GitHub's changelog feature to fill out the release message.
